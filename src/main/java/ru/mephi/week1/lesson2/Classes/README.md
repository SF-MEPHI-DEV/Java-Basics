## Аннотация @Override

**@Override** - это аннотация компилятора, которая:

1. **Проверяет корректность переопределения** - компилятор убедится, что метод действительно переопределяет метод родительского класса или интерфейса
2. **Защищает от опечаток** - если вы случайно напишете неправильное имя метода, компилятор выдаст ошибку
3. **Улучшает читаемость кода** - явно показывает, что метод переопределяется
4. **Помогает при рефакторинге** - если изменится сигнатура метода в родительском классе, компилятор укажет на проблему

**Примеры использования в коде:**
- `Animal.java` - абстрактные методы переопределяются в наследниках
- `Dog.java:20, 25, 32, 38, 59` - переопределение методов Animal
- `RegistrationRequestDto.java:49` - переопределение toString()
- `Latitude.java:17`, `Longitude.java:17` - переопределение toString()
